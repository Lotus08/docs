API stands for application programming interface.
You can read more about APIs at https://www.redhat.com/en/topics/api/what-are-application-programming-interfaces.

To send notifications to your users, use MagicBell's REST API.
We also offer wrappers for some programming languages that simplify the
communication with our API. You can read the complete REST API reference
at https://magicbell.com/docs/rest-api/reference.
